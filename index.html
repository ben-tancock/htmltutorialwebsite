Tables

The table element is used to create a table. The contents of the table are written out row by row. You indicate the start of each row with the tr tag. The tr stands for Table Row. Inside each tr tag is multiple td tags. Let's try it out below:

15 20 25
30 60 73

We can also use the th tag which is used to represent the heading for either a column or a row. It is declared inside the tr tag, and the scope of a th tag is specified inside the tag as a variable as either "row" or "col". Note that when we make the th tags for the columns, we need to make one empty th tag to align the column headers properly.

Saturday Sunday
PeePee: 12 34
PooPoo: 15 23

Spanning Rows and Columns:

Sometimes you may need entries in a table to stretch across more than one column. The colspan attribute can be used on a th or td element and indicates how many columns that cell should run across.

9am 10am 11am 12am
Monday Geography Math Art
Tuesday Gym Home Ec

It's sorta hard to see, but we used colspan to make Geography take up 2 cells and Gym take up 3. Let's try and pretty this up with some styling (later)!

Long Tables:

There are 3 elements that help distinguish between the main content of the table and the first and last rows.

These elements help people who use screen readers and also allow you to style these sections in a different manner
than the rest of the table (as you will see when you learn about css)

These elements are thead, tbody, and tfoot.

thead: the headings of the table should sit inside the thead element.
tbody: the body should sit inside the tbody element.
tfoot: the footer belongs inside the tfoot element.

In this example, Date, Income and Expenditure are inside the thead tag, even though they aren't the only headers, they are the column headers of the table.

Everything from the 1st of January to the 31st of January is in the tbody tag.

The numbers at the bottom, 7824 and 1241 are in the tfoot tag.
